| Параметр | Описание                          | Пример                     | Типизация              | Дефолтное |
|----------|-----------------------------------|----------------------------|------------------------|-----------|
| `map`    | [**обязательный**] Словарь замены типов контента | `{"*/*": "application/json"}` | `Record<string, string>` | `{}`        |

Пример конфигурации:

```js
module.exports = {
    pipeline: [
        // ... other rules
        {
            rule: "change-content-type",
            config: {
                map: {
                    "*/*": "application/json"
                }
            },
        }
        // ... other rules
    ]
}
```
