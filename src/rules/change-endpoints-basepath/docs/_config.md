| Параметр    | Описание                          | Пример                     | Типизация              | Дефолтное |
| -------- |-----------------------------------|----------------------------|------------------------|-----------|
| `map`  | [**обязательный**] Словарь замены | `{ '/public/api': '' }` | `Record<string, string>` | `{}`        |


Пример конфигурации:

```js
module.exports = {
    pipeline: [
        // ... other rules
        {
            rule: "change-endpoints-basepath",
            config: {
               map: { 
                   '/public/api': '',
               }
            },
        }
        // ... other rules
    ]
}
```
